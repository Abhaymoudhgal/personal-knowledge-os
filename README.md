# Personal Knowledge Operating System (PKOS)

A Personal Knowledge Operating System that combines:

- Retrieval Augmented Generation (RAG)
- Memory
- Knowledge Graphs
- AI-powered Question Answering

## Vision

Enable users to chat with their documents, notes, resumes, research papers, and technical knowledge.
